General Info
LDapper is an LDAP client for searching and browsing LDAP servers.
About LDapper Show

LDapper is an LDAP (Lightweight Directory Access Protocol) client for finding email addresses. You can look up people by name, department, or some other arbitrary attribute. Once you have a list of people, you can copy/paste or drag addresses into a new email message. You can probably automagically add them to a new email message by choosing the "Mail To" menu. You can export addresses directly into your Mac's Contacts (aka Address Book) or files that can be imported into other applications. With LDapper, you can also browse an LDAP server in a hierarchical fashion.

Unlike older versions of LDapper which used the LDAP framework (or OpenLDAP libraries) directly, this version is really a "wrapper" around the built-in LDAP utility ldapsearch that is included by default with Mac OS X. You can think of LDapper as a GUI "front-end" to ldapsearch. Many of the options and preferences you specify simply modify the command line arguments to the ldapsearch command. This version of LDapper uses separate preferences than previous versions so you can continue to use an older version if you wish. The first time you run this version, LDapper will import any directories from the old version. Any subsequent changes to those directories will only affect the new version. LDapper now requires Mac OS X 10.7 (Lion) or later.

The latest version of LDapper can be found here. (http://carl-bell.baylor.edu/carl_bell/stuff.html)

Release Notes Show

3.1.1 (02-Apr-2013)

  • Saving current search updates the saved searches window.
  • Saving current search now uses the name "New Search" instead of the name of the previous search.
  • Focus ring around search criteria in a Search window is no longer visible when running on 10.7.
  • Tweaked initial position/size of windows a bit.

3.1 (12-Feb-2013)

  • Now requires Lion (10.7) or later.
  • Now signed with DeveloperID so should launch with strict(er) Gatekeeper settings.
  • Changed how user specifies search criteria. Can now add up to 4 criteria by clicking (+) and (-) buttons.
  • Can now save searches. Works somewhat like a web browser's "bookmarks".
  • No longer supports a directory's "Other" attribute. Instead, user specifies the attribute when choosing "Other..." popup menu item in the Search window.
  • Description attribute(s) added to note when exporting to Contacts.
  • Will use displayname attribute for first name if givenname doesn't exist when exporting to Contacts.
  • Added printing support.
  • Renamed cold prickly "Info Text" references to warm fuzzy "Details".
  • Although the Details and Command Output text fields are not editable, you can still Cut and Clear any selected text (and/or images).
  • Can now export details for all entries as well as selected entries.
  • LDapper documents are no longer supported.
  • Help menu now uses standard Mac Help Center.
  • Tweaked the default (and minimum) window size a bit.

3.0 (22-Jun-2012)

  • Initial non-beta release of "3.0" version.

3.0b1 (26-Apr-2012)

  • Initial beta release of "3.0" version.
